
## Let, Var and Const

| Compare                        | let   | const | var                      |
|  ---                           | ---   | ---   | ---                      |
|Scope                           | block | block | function                 |
|Use before declaration location | error | error | value will be undefined  |
|Re-declare in function/Scope    | error | error | will work                |
|Re-assign value                 | yes	 | error | yes                      |
